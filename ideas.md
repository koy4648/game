# 프로포즈 사이트 디자인 아이디어

## 아이디어 1: 러브레터 스크랩북

<response>
<text>
**Design Movement**: 빈티지 스크랩북 + 현대적 귀여움 (Cottagecore meets Y2K)

**Core Principles**:
- 손으로 쓴 듯한 텍스트와 스티커 감성
- 따뜻한 크림/베이지 배경에 알록달록한 포인트
- 사진 폴라로이드 스타일 프레임
- 종이 질감과 테이프 장식 요소

**Color Philosophy**:
- 배경: 크림 (#FFF8F0), 따뜻한 베이지
- 포인트: 코랄 핑크 (#FF6B8A), 민트 (#7EC8C8), 레몬 옐로우 (#FFE566)
- 텍스트: 다크 초콜릿 (#3D2B1F)
- 감정: 따뜻하고 향수로운, 손으로 만든 사랑의 느낌

**Layout Paradigm**:
- 스크랩북 페이지처럼 요소들이 살짝 기울어진 비대칭 배치
- 스티커, 테이프, 손글씨 장식이 곳곳에 흩어진 레이아웃
- 페이지 넘기는 애니메이션

**Signature Elements**:
- 폴라로이드 스타일 카드
- 마스킹 테이프 장식
- 손글씨 폰트 혼용

**Interaction Philosophy**: 스크랩북 페이지를 넘기듯 진행, 스티커를 클릭하면 반응

**Animation**: 페이지 플립 효과, 스티커 흔들림, 하트 파티클

**Typography System**:
- 제목: Gaegu (한국어 손글씨 느낌)
- 본문: Noto Sans KR

## 아이디어 2: 별빛 동화 여행 ✨ (선택)

<response>
<text>
**Design Movement**: Dreamy Fairytale Illustration + Soft Pastel Romanticism

**Core Principles**:
- 밤하늘과 별빛을 모티프로 한 몽환적 분위기
- 귀엽고 발랄하면서도 로맨틱한 균형
- 두 사람의 여행(소개팅→연애→프로포즈)을 별자리 지도처럼 표현
- 부드러운 그라디언트와 반짝이는 파티클

**Color Philosophy**:
- 배경: 딥 네이비 (#1A1A3E) → 미드나잇 퍼플 (#2D1B69) 그라디언트
- 포인트: 골드 (#FFD700), 소프트 핑크 (#FFB3C6), 라벤더 (#C8A8E9)
- 별빛: 화이트 (#FFFFFF), 크림 (#FFF9E6)
- 감정: 꿈같고 로맨틱한, 마치 동화 속 이야기처럼

**Layout Paradigm**:
- 세로 스크롤 스토리텔링 (별자리를 따라가는 여정)
- 각 섹션이 별과 별 사이를 이동하는 느낌
- 중앙 집중이 아닌 좌우 교차 배치

**Signature Elements**:
- 반짝이는 별 파티클 배경
- 별자리 연결선으로 표현된 추억들
- 달과 별 아이콘

**Interaction Philosophy**: 별을 클릭하면 추억이 펼쳐지는 인터랙션

**Animation**: 별 반짝임, 유성 효과, 페이드인 스크롤 애니메이션

**Typography System**:
- 제목: Gowun Batang (우아한 한국어)
- 본문: Noto Sans KR
</text>

## 아이디어 3: 귀여운 픽셀 어드벤처 게임

<response>
<text>
**Design Movement**: Pixel Art RPG + Kawaii Pop

**Core Principles**:
- 레트로 픽셀 게임 UI 스타일
- 두 사람의 이야기를 RPG 퀘스트처럼 표현
- 귀여운 캐릭터 스프라이트
- 게임 진행 바와 경험치 시스템

**Color Philosophy**:
- 배경: 연한 하늘색 (#E8F4FD), 민트 그린 (#D4F5E9)
- UI: 파스텔 핑크 (#FFD6E0), 파스텔 퍼플 (#E8D5F5)
- 포인트: 비비드 핑크 (#FF69B4), 선명한 민트 (#00CED1)
- 감정: 발랄하고 재미있는, 게임처럼 즐거운 프로포즈

**Layout Paradigm**:
- 게임 화면처럼 고정된 뷰포트
- 상단 HP/MP 바 스타일의 진행 표시
- 대화창 스타일의 텍스트 박스

**Signature Elements**:
- 픽셀 아트 스타일 아이콘
- 게임 대화창 UI
- 레벨업 이펙트

**Interaction Philosophy**: 게임 컨트롤러처럼 버튼 클릭으로 진행

**Animation**: 픽셀 파티클, 레벨업 이펙트, 바운스 애니메이션

**Typography System**:
- 제목: DungGeunMo (픽셀 폰트)
- 본문: Noto Sans KR
</text>
